WHAT ARE PARALLEL ROUTES?
â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€

Parallel routes allow you to simultaneously render multiple independent pages
in the same layout. Each "slot" (prefixed with @) can have its own routing,
content, and state while appearing side-by-side in the same layout.

KEY DIFFERENCE FROM INTERCEPTING ROUTES:
- Parallel Routes: Render multiple content streams SIDE-BY-SIDE
- Intercepting Routes: Intercept navigation and REPLACE default content
- Can use both together!
